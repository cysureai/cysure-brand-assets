# cysure-brand-assets

Assets públicos de marca Cysure servidos por URL raw (covers de Notion, LinkedIn, Drive).

Uso: URL raw del archivo, p. ej.
`https://raw.githubusercontent.com/cysureai/cysure-brand-assets/main/<archivo>`

Fuente canónica de diseño: skill `cysure-brand` (Design System v4.1, logo v2.1 Jul 2026).

Archivos:
- `cysure-mark.svg` / `cysure-lockup.svg`: vectores oficiales del mark y del lockup (v2.1).
- `cysure-mark-firma-1024.png`: mark sobre blanco, para firma electrónica.
- `google-workspace-logo-320x132.png`: lockup horizontal para Google Workspace.
- `cover-notion-linkedin-drive-blanco-2400x600.png`: cover claro para Notion, LinkedIn y Drive.
